# Landbruksdirektoratet Gjødselsregister API
API for systemvareleverandører som skal integrere seg mot Landbruksdirektoratets Gjødselsregister. Du kan lese mer om prosjektet her på Landbrukdsdirektoratet sine nettsider: https://www.landbruksdirektoratet.no/nb/jordbruk/miljo-og-klima/husdyrgjodsel-og-gjodsling/utvikling-av-digitale-stotteverktoy-for-oppfolging-av-gjodselbrukforskriften

## Bakgrunn for prosjektet
Utvikle systemer som effektivt støtter gjødselbrukere å etterleve det nye gjødselbrukregelverket https://lovdata.no/dokument/SF/forskrift/2025-01-29-115.

# API beskrivelse
Dette API-et gir tilgang til å sende og motta meldinger og søkander med forvaltningen av Gjødselsregisteret for at gjødselsbruker skal kunne rapportere pålagte meldinger og søknader. 

## Samhandling om videreutvikling av API
Vi inviterer eksterne systemvareleverandører til samarbeid om et felles API som forvaltes i dette GitHub-repositoriet. All diskusjon, avklaringer og innspill skal skje i Issues. Opprett én issue per tema, bruk labels (f.eks. proposal, question, bug), og referer til relevante spesifikasjoner/krav. Endringsforslag leveres som Pull Requests som refererer til en issue; PR-er skal beskrive motivasjon, konsekvenser for bakoverkompatibilitet og inkludere oppdaterte tester og dokumentasjon.
